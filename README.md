# stm32_cube_test2
Test project for platformio with stm32cube framework using STM32CubeMX generated code.
FreeRTOS is enabled and uses STM32Cube Middleware-FreeRTOS library.
The project writes test messages to UART3(STLINK) and to the USB OTG port (virtual serial port device).

## Info
* Board: nucleo_f446ze
